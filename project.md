# QA Projects

## API Automation Framework

Java와 Rest Assured 기반으로 API 테스트 자동화 프레임워크를 구축한 프로젝트입니다.

### Tech Stack

- Java
- Rest Assured
- TestNG
- Maven

### Features

- REST API 테스트 자동화
- Response Validation
- 테스트 리포트 생성
- CI 환경에서 실행 가능
