## About Me

저는 **테스트 자동화와 API 테스트를 중심으로 품질을 개선해 온 QA 엔지니어**입니다.
2016년부터 다양한 서비스와 시스템에서 QA 업무를 수행하며 **테스트 전략 수립, 테스트 자동화 구축, API 테스트 환경 설계**를 경험했습니다.

특히 단순히 테스트를 수행하는 것을 넘어 **테스트 자동화를 통해 품질 검증을 효율적으로 만들고, 개발 프로세스 안에서 품질을 지속적으로 확인할 수 있는 구조를 만드는 것**에 집중해 왔습니다.

Java 기반 테스트 자동화와 API 테스트를 활용하여 반복적인 검증 작업을 자동화하고, CI 환경에서 안정적으로 실행될 수 있도록 테스트 프레임워크를 구축하는 경험을 쌓아왔습니다.

앞으로도 **품질을 개발 과정의 마지막 단계가 아닌, 개발 과정 전체에서 지속적으로 검증되는 구조로 만드는 QA 엔지니어**로 성장하는 것을 목표로 하고 있습니다.
